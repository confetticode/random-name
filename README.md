## Requirements

- PHP ^8.0

## Installation

You can install the package via composer:

```bash
composer require confetticode/random-name
```

## Usage

```php
<?php

require __DIR__ . '/vendor/autoload.php';

echo random_name(); // Eg: golick-suspicious
```
